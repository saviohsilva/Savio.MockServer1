# Project Brief — Savio Mock Server

## O que é

Savio Mock Server é uma aplicação web Blazor Server que permite criar, gerenciar e servir mocks de APIs HTTP em tempo real. Funciona como um servidor de mocks pessoal com interface visual completa.

## Objetivo Principal

Substituir ferramentas externas de mock (Mockoon, WireMock, etc.) com uma solução self-hosted, multi-usuário, com interface web acessível de qualquer lugar.

## Problema que Resolve

Desenvolvedores precisam de endpoints HTTP falsos para testar integrações. As soluções existentes exigem instalação local, não têm interface web ou não suportam múltiplos usuários isolados.

## Proposta de Valor

- Cada usuário tem um alias único → URLs de mock isoladas por usuário
- Interface visual completa → sem necessidade de editar arquivos JSON/YAML
- Self-hosted → dados e configurações sob controle do próprio usuário
- Histórico de requisições → visibilidade completa das chamadas recebidas
